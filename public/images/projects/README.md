# Project Screenshots

This directory contains screenshots for each project displayed in the portfolio.

## Naming Convention

Screenshots should be named using kebab-case matching the project title:

- `jose-madrid-salsa.png` - Jose Madrid Salsa project
- `amplinks.png` - Amplinks project
- `zanesville-store.png` - Zanesville.store project
- `homesh-app.png` - Homesh.app project
- `apple-sider.png` - Apple-Sider project
- `world-auto-net.png` - World Auto Net project
- `neff-paving.png` - Neff Paving project
- `first-baptist-church.png` - First Baptist Church project
- `ohio-interests.png` - Ohio Interests project

## Image Specifications

- **Format**: PNG or JPG
- **Recommended dimensions**: 1200x900px (4:3 aspect ratio)
- **Maximum file size**: 500KB (optimize for web)
- **Content**: Full homepage screenshot or hero section
- **Quality**: High quality, clear and professional

## How to Add Screenshots

1. Take a screenshot of the project's homepage
2. Crop to focus on the hero/main section
3. Optimize the image (use tools like TinyPNG or Squoosh)
4. Save with the correct filename in this directory
5. The image will automatically appear in the top right corner of the project card

## Display Info

Screenshots are displayed:
- **Position**: Top right corner of project card
- **Sizes**: 
  - Mobile: 128x96px
  - Tablet: 224x160px
  - Desktop: 288x192px
- **Styling**: Rounded corners, white border, shadow
